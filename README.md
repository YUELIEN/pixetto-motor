
# Pixetto 與馬達控制 Mind+ 用戶庫 (v0.0.2)

本用戶庫專為 Mind+（上傳模式）設計，讓使用者能透過 Pixetto 視覺感測器進行顏色辨識或神經網路辨識，並結合循線感測器來精準控制雙馬達運作。

## 📦 載入方式

### 本地載入

1. 下載本專案提供的 `yuelien-pixetto_motor_control-thirdex-V0.0.2.mpext` 檔案。
2. 開啟 Mind+ → 點選左下角「擴展」 → 「用戶庫」 → 「從本地載入」。

### 網路載入

1. 開啟 Mind+ → 「擴展」 → 「用戶庫」。
2. 在搜尋欄貼上專案網址：
`https://github.com/YUELIEN/pixetto-motor`

## 🔌 硬體接線配置 (v0.0.2 標準)

本庫預設腳位配置同時相容於 **mBot mCore** 控制板與 **標準 Arduino** 接線。

### 1. mBot mCore 連接埠對應

| 模組 | mCore 連接埠 | Arduino 內部腳位 |
| --- | --- | --- |
| **循線感測器** | 連接埠 2 | Slot1=D9 (左), Slot2=D10 (右) |
| **Pixetto 感測器** | 連接埠 3 | Slot1=A2 (RX), Slot2=A3 (TX) |
| **右馬達 (M1)** | 馬達 M1 | D6 (PWM), D7 (DIR) |
| **左馬達 (M2)** | 馬達 M2 | D5 (PWM), D4 (DIR) |

### 2. 標準 Arduino 獨立接線

* **Pixetto 感測器**：TX -> A2, RX -> A3。
* **循線感測器**：左感測器 -> Pin 9, 右感測器 -> Pin 10。
* **馬達驅動 (如 L298N)**：
* 左馬達：Pin 4 (方向), Pin 5 (速度/PWM)。
* 右馬達：Pin 6 (速度/PWM), Pin 7 (方向)。



## 🧱 積木功能說明

| 積木類型 | 積木名稱 | 功能說明 |
| --- | --- | --- |
| **設定** | 初始化 Pixetto 感測器 | 啟動通訊並自動設定馬達與感測器腳位。 |
| **偵測** | 設定 Pixetto 偵測功能 | 可切換「顏色偵測」或「神經網路辨識」模式。 |
| **判斷** | 偵測到 [顏色/ID] ？ | 判斷是否辨識出特定目標（支援紅/黃/綠與 NN ID）。 |
| **判斷** | 循線感測器偵測到 [位置] | 判斷左/右/全部是否遇到黑線或白線。 |
| **動作** | 前進/後退/左轉/右轉 | 控制動力輸出（0-100%），內部自動換算 PWM。 |
| **動作** | 停止 | 立即停止雙輪馬達。 |

## 💡 範例邏輯：紅燈停、綠燈行

```text
初始化 Pixetto 感測器
設定 Pixetto 偵測功能為 [顏色偵測]
重複無限次：
  如果 Pixetto感測器 偵測到 [紅色] ？：
    停止
  否則 如果 Pixetto感測器 偵測到 [綠色] ？：
    前進，動力 50%
  否則：
    後退，動力 20%

```
